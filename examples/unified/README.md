# Unified SDK Example

Use ModelScan's intelligent router for automatic provider selection.

## Setup

```bash
export OPENAI_API_KEY="your-openai-key"
export ANTHROPIC_API_KEY="your-anthropic-key"
```

## Run

```bash
go run main.go
```

## Features

- Automatic provider selection
- Health-based routing
- Rate limit awareness
- Fallback on failure
- Cost optimization
