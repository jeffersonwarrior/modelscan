# Basic Example

Simple usage of ModelScan with OpenAI provider.

## Setup

```bash
export OPENAI_API_KEY="your-api-key"
```

## Run

```bash
go run main.go
```

## Expected Output

```
Found 24 OpenAI models
Example model: gpt-4o (GPT-4o)
  Pricing: $2.50/$10.00 per 1M tokens
  Context: 128000 tokens
```
